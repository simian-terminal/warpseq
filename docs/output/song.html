Song

This chapter will explain some terms in Warp, and compare them a bit to what you may expect with audio production tools you may be familiar with.

  • Each Warp file is called a song. Each song definition is self contained. There is no "computer specific" setup in Warp.
  • A song must define a default tempo, though this may be overriden in a Scene or Clip. The default is 120. Change it.
  • A song may define a default Scale, though this may also be overriden in a Scene or Clip.
  • Songs can be saved and loaded to files, and files should be openable by any future version of Warp.
  • Songs are technically JSON files, but you hand edit them at your own risk. If that seems appealing, see the Python API

  If you don't want to set up each Device or Instrument each time, a good idea is to make an empty Warp song called "template.warp". This is also a good place to keep your favorite user Scale definitions.
