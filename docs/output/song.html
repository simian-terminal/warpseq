Song

This chapter will explain some terms in Warp, and compare them a bit to what you may expect with music production tools you may be familiar with.

  • Each Warp file is called a Song. Each song definition is self contained in terms of configuration, so there is no equivalent to a "setup" or "preferences" menu in Warp.
  • A song must define a default tempo, which may be overriden in a Scene or Clip. The default is 120. Change it.
  • A song may define a default Scale, which may be overriden in a Scene or Clip.
  • Songs can be saved to and loaded from files, and files should be compatible with any future version of Warp.
  • Songs are technically JSON files, but hand edit them at your own risk. Instead, for working with songs programmatically, see the Python API

  If you don't want to set up each Device or Instrument each time, a good idea would be to make an empty Warp song called "template.warp" and start projects by opening this file. This is also a good place to keep your favorite user Scale definitions.
