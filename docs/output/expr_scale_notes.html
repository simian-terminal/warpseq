Scale Notes & Chords

Now that you know how to input specific Literal Notes, it is time to learn how to input notes that will respond to scale/mode changes.

  Most DAWs implement scales with "force to scale" semantics, which can result in confusion when scales notes in the input scale do not line up cleanly with the output scale, and makes it difficult to tell the scale degrees of notes when looking at them. Also, these systems do not allow input of accidental notes (sharps and flats). Warp handles this totally differently, solving all of these problems.

Warp uses lists of Arabic numerals to indicate degrees along a scale, and Roman numerals to represent basic chord types. We'll show you how to access more complex chords as well, using Mod Expressions.

A Scale Example

Here is the same scale mentioned in Literal Notes, written in a way that will instantly work for any 7-note Scale: 
1
2
3
4
5
6
7

That was simple - it's just the scale degrees for each note we want to play. The Scale value will be taken from the clip, if so provided, or if not, can come from the Scene, and if not set there, will use the value from the Song. If you don't set a scale anywhere, the system will assume the C Major scale. Users do not even need to know what notes are in the scale, because Warp will handle all the music theory for you.

Scale degrees may also be listed as negative, which shift down from the root, as shown in this sequence: 

1
-2
4
5
-3
6
-4
1
Note: The value 0 is not valid here, because we don't know what it would mean. In that sense, it's like Math before the 12th century.

Roman Numeral Chords

Chords can be expressed as Roman numerals: 
I
IV
V

This notation is already commonly used in music to denote chord progressions. Like the arabic numerals above, this system is nice because it doesn't care what scale you are in.

Lowercase roman numerals are used to represent minor chords: 
i
iv
v
More advanced types can still be represented in Warp using their base notes and chord type: 
1:sus4
2:major
4:power
Or, equivalently using Mod Expressions: 
1 ch=sus4
2 ch=major
4 ch=power
For reference the built-in chord types you can use this way are:
  • minor
  • major
  • dim
  • aug
  • sus4
  • sus2
  • fourth
  • power
  • fifth
  • M6
  • m6
  • dom7
  • M7
  • m7
  • aug7
  • dim7
  • mM7
  Chords and notes can be mixed together in the same lists. Chords can also be modified by all Mod Expressions exactly like notes can.
