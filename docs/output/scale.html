Scales

A scale represents a named musical scale, having a scale type (aka "mode"), and starting from a certain note (aka "root"). In general, Warp suggests defining notes and chords as relative scale intervals and chords versus literal notes, except for percussion tracks. Scales are fundamental to Warp as the program was largely created to improve upon the "force to scale" paradigm commonly used in other tools.

  • Warp ships with many pre-defined MIDI scale types
  • Users can also define their own scales by typing in a list of interval symbols: 1, 2, b3 ("flat third"), 5 ("perfect fifth"), etc.
  • Scale names used in patterns are symbolic links to the scale definition, so when you edit the parameters of a named scale (like "intro1"), they change the scale everywhere that name is used.
  • A scale may be set on the Song, or overridden on the Scene or Clip.

 

Built in scale types include:

  • major (aka: ionian)
  • pentatonic
  • pentatonic_minor
  • natural_minor (aka: aeolian, or just "minor")
  • blues
  • dorian
  • chromatic
  • harmonic_major
  • harmonic_minor
  • locrian
  • lydian
  • major_pentatonic
  • melodic_minor_asc
  • melodic_minor_desc
  • minor_pentatonic
  • mixolydian
  • phyrigian
  • japanese
  • akebono

More built-in scales will be added in future versions
