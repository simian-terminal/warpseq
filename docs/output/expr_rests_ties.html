Silence (Rests)

Often we don't want a pattern to be constantly playing notes. We might want to leave some gaps.

An "x" or an empty space represents a slot in a pattern where nothing should play.

"x" can be a mod expression following a note, that can silence the note without having to delete it from the pattern.

  If all your notes are the same length, you can just adjust the rate on a pattern or clip to slow it down without using ties.
  The Warp system does not currently allow for holding one note in a chord longer than another on a single track, or playing one note slightly later - but arpeggiators exist via Transforms, if you need to do this, you can have multiple tracks target the same instrument, and that is an easy way to accomplish this.
  A format for dotted note durations (kind of like a half tie, using "." to follow the note) will probably be added in the near future.
