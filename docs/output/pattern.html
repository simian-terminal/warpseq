Patterns

A pattern represents an ordered series of Expressions that evaluate into notes or chords when used inside a Clip.

  • Patterns include strings like:
    • "1"
    • "IV"
    • "C4sus4"
    • or even "C4sus4;O+1;cc0=50".
  • Patterns can include velocity and MIDI CC information (see Modifer Expressions.
  • A pattern may define an octave_shift to be applied to the current Scale, which applies after the base octave of the instrument.
  • A pattern may define a Tempo to override the tempo of the Scene or Song, if not overriden by the Clip.
  • A pattern may define a Scale to override the scale of the Scene or Song, if not overriden by a Clip.
